# Frontend App (Vite + React + TypeScript)

This folder contains the web client for SSWI.

## Scripts

```bash
npm run dev        # Start Vite dev server
npm run build      # Create production build in dist/
npm run preview    # Preview production build locally
npm run test       # Run unit tests once
```

## Key Folders

- `src/pages/`: Route-level pages
- `src/components/`: Reusable UI and layout components
- `src/lib/`: API clients and utility modules
- `src/store/`: App state store
- `src/test/`: Frontend test suite
- `public/`: Static assets copied as-is

## Notes

- Do not edit `dist/` files directly; they are generated by Vite.
- Keep app logic under `src/` to preserve clean build boundaries.
